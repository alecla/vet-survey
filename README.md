# vet-survey
A pet clinic survey. Built with Typescript and React.

## Prerequisities
Requires Node.js 16+ 

## Dev

``` bash
# install dependencies
npm i

# dev
npm run dev
```

Visit [http://localhost:5173/](http://localhost:5173/)

## Build

``` bash
# install dependencies
npm i

# build
npm run build

# preview
npm run preview
```

Visit [http://localhost:8888/](http://localhost:8888/)

Enjoy!
